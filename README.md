## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/todo-app.git
```

Go to the project directory

```bash
  cd ..todo-app
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run start
```
